# CSharpCalc

A Windows Forms calculator application built in C# - my first C# project.

![Preview](https://github.com/DavoDC/CSharpCalc/blob/main/Preview.png)

## Features

- Standard arithmetic operations (add, subtract, multiply, divide)
- Clean Windows Forms interface
- Keyboard and mouse input support

## Running

Open in Visual Studio and build, or run the compiled executable directly.

## Tech

- **Language:** C#
- **Framework:** Windows Forms (.NET)
- **IDE:** Visual Studio

## Development

**December 2022** · Built as an introduction to C# and the Windows Forms UI framework.
